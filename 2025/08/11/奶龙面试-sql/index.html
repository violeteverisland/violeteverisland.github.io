发表于 2025-08-11

创建表 stu

情况 1：无索引

查询： SELECT id, score, description FROM stud WHERE name > 'yuanzhe' AND score < 400;

情况 2：有索引

索引：

  • id (主键)
  • name_idx (name)
  • score_idx (score)

查询： SELECT id, score, description FROM stud WHERE name > 'yuanzhe' AND score < 400;

情况 3：LIKE查询与索引

索引：

  • id (主键)
  • name_idx (name)
  • score_idx (score)

查询： SELECT id, score FROM stud WHERE name LIKE 'yuanzhe%' AND score < 800;

情况 4：索引覆盖

索引：

  • id (主键)
  • name_idx (name)
  • score_idx (score)

查询： SELECT id, score FROM stud WHERE score >= 100;
